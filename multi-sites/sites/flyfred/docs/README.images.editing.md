#

## Receita que costuma funcionar muito bem no GIMP

### 1) Saturation

**Colors > Hue-Saturation**

* **Saturation: -15**

### 2) Contrast

**Colors > Brightness-Contrast**

* **Contrast: -10**

### 3) Brightness

* **Brightness: +5**

### 4) Opcional: camada branca suave 🤍

* Nova layer branca por cima
* **Opacity: 5% a 10%**

➡️ Isso tira aquele aspecto muito “cru” da foto.

## Resultado:

* menos viva
* mais elegante
* mais integrada ao design
* sem parecer apagada

## Regra importante 🎯

Se passar de:

* Saturation abaixo de -25
* White layer acima de 15%

➡️ começa a parecer lavado demais.

💡 Muito designer usa exatamente isso para deixar imagens com cara de site moderno / apresentação profissional.
