Tipografia

Duas famílias tipográficas extraídas do HTML. Ambas via Google Fonts.

Preview das fontes em uso
Plus Jakarta Sans — Títulos & Display
Inter — corpo de texto, legendas e elementos de suporte

2.1 Fontes utilizadas

Papel Família Token CSS Pesos usados Google Fonts?
Títulos, subtítulos, botões, labels, navbar Plus Jakarta Sans --font-display 400 · 500 · 600 · 700 · 800 Sim
Corpo de texto (p, descrições, notas) Inter --font-body 400 · 500 · 600 Sim

2.2 Escala tipográfica

Elemento Família Tamanho Peso Line-height Letter-spacing Notas
H1 — Hero headline Plus Jakarta Sans clamp(32px, 5vw, 54px) 800 1.1 −0.02em Fluido. Mobile ≈ 32px, desktop ≈ 54px.
H2 — Section title Plus Jakarta Sans clamp(28px, 4vw, 40px) 800 1.15 normal Classe .section-title. Mobile ≈ 28px, desktop ≈ 40px.
H3 — Step title / card title Plus Jakarta Sans 18px 700 1.3 normal Títulos de cards e etapas do "Como Funciona".
Parágrafo / body Inter 17px 400 1.7 normal Classe .section-sub. Texto corrido das seções.
Body menor / apoio Inter 15px 400 · 500 1.6 normal Descrições de cards, items de lista, texto de widget.
Label / kicker / badge Plus Jakarta Sans 12px 700 1 0.06–0.10em Classes .section-label e .badge. Sempre uppercase.
Botão (CTA / ghost) Plus Jakarta Sans 16px 700 1 normal Classe .btn. Padding: 16px 32px. Botão WPP: 15px.
Navbar link Plus Jakarta Sans 13px 600 1 normal Oculto em mobile (max-width: 600px).
Navbar logo Plus Jakarta Sans 22px 800 1 normal Logotipo textual. Parte branca + parte coral.
Caption / small Inter 13px 400 · 500 1.5 normal Notas legais, horários, rodapé de cards.
Footer corpo Inter 14px 400 1.6 normal Links e texto do footer.
Responsividade tipográfica: H1 e H2 usam clamp() — valores fluidos entre mobile e desktop. Os demais elementos têm tamanho fixo em px, mas o font-size: 16px no html serve de base para o sistema rem. A escala clamp() foi escolhida para eliminar breakpoints de tipografia desnecessários.
Seção 3

Google Fonts — link de importação

Link extraído diretamente do HTML original. Inclui todos os pesos em uso.

<!-- Cole no <head> do projeto, antes de qualquer CSS -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@400;500;600;700;800&family=Inter:wght@400;500;600&display=swap" rel="stylesheet">
Para Tailwind CSS v4: Adicione ao theme.css as variáveis de fonte: 
@theme {
  --font-display: 'Plus Jakarta Sans', sans-serif;
  --font-body: 'Inter', sans-serif;
}
